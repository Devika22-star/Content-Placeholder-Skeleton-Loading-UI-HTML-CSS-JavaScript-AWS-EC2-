# Content-Placeholder-Skeleton-Loading-UI-HTML-CSS-JavaScript-AWS-EC2-
Developed a skeleton loading UI using HTML, CSS, and JavaScript to improve user experience during content loading. Designed animated placeholder elements (image blocks and text bars) using CSS keyframe animations. Implemented JavaScript logic to simulate loading delay and dynamically switch from skeleton screen to actual content. 
Skeleton Loading UI Project

Project Description

This project demonstrates a Skeleton Loading UI, which improves user experience by showing placeholder content while actual data is loading.

Instead of displaying a blank screen, users see animated grey boxes (skeletons), giving a smoother and more engaging experience.

---

Features

- Skeleton loading animation using CSS
- Dynamic content loading using JavaScript
- Clean and responsive UI
- Improves perceived performance

---

Technologies Used

- HTML
- CSS
- JavaScript
- AWS EC2 (for deployment)

---

How It Works

1. Initially, skeleton UI (grey placeholders) is displayed.
2. JavaScript delays content loading for a few seconds.
3. After loading, real content replaces the skeleton UI.

---

How to Run

1. Download or clone the project
2. Open "index.html" in any browser

---

Deployment

The project is deployed using AWS EC2:

- Launched EC2 instance
- Installed Apache server
- Uploaded project files
- Configured security group for HTTP access

---

Output

- First shows loading skeleton
- Then displays actual content after delay

---

Use Case

- Used in modern applications like Instagram, YouTube
- Enhances user experience during slow network/data loading
